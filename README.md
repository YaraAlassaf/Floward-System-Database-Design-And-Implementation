# Floward System Database Design and Implementation

## Project Overview

This project presents the complete database design and implementation for Floward, an online flower and gift delivery platform. The work covers all fundamental phases of database development, including conceptual design, relational schema creation, SQL implementation, data manipulation, and query execution.

The project was developed as part of the Database Fundamentals course at Princess Nourah bint Abdulrahman University.

## Project Features

- Entity Relationship Diagram (ERD)
- Relational Schema Design
- Database Normalization
- SQL DDL Commands
- SQL DML Commands
- Sample Data Insertion
- SQL Queries and Results
- Database Constraints
- Primary and Foreign Key Relationships

## Technologies Used

- SQL
- Relational Database Design
- ERD Modeling
